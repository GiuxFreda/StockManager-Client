# StockManager – Client

Questo repository contiene **solo l'installer pubblico** dell'applicazione **StockManager** per Windows.

L'app è pensata per piccoli negozi che vogliono gestire in modo semplice:

- prodotti e varianti
- giacenze di magazzino
- vendite e scontrini
- statistiche di base su vendite e stock

---

## Requisiti

- **Sistema operativo:** Windows 10 o successivo (64 bit)
- Non è necessario avere Java installato: è incluso nell'installer.

---

## Download

1. Vai alla sezione **Releases** di questo repository.
2. Scarica il file `.msi` più recente (es. `StockManager-1.1.0.msi`).
3. Salva il file sul tuo PC.

Link diretto alle release:  
https://github.com/GiuxFreda/StockManager-Client/releases/latest

---

## Installazione

1. Fai doppio clic sul file `.msi` scaricato.
2. Segui i passaggi guidati dell'installazione (Avanti → Avanti → Fine).
3. Al termine, troverai **StockManager** nel menu Start e/o sul Desktop.

---

## Primo avvio

Al primo avvio puoi:

- creare o importare un negozio
- iniziare a inserire i prodotti
- impostare categorie e varianti
- registrare le prime vendite

I dati vengono salvati in locale sul tuo PC.
